默认

                    
     <lr-search :auto="auto" @search="search" @clear="clear" ></lr-search>
props参数说明
参数 说明 类型 可选值 默认值
auto 搜索框的搜索建议数据源 array -- --
事件说明, 在非作用域插槽 的情况下可用
事件 说明 回调参数
autoSelected 搜索建议列表 选中事件 返回当前选中行的数据
clear 清空搜索框内容 无返回
search 搜索按钮 返回 搜索框的值
focus 点击输入框事件 返回 搜索框的值
slot使用
name 说明
无 直接在组件内部写,用于左侧内容显示
