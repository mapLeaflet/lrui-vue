props参数说明
参数 说明 类型 可选值 默认值
body 是否展示内容面板 Boolean true , false true
bottom 在底部展示内容面板,直接写在 控件属性上 Boolean -- --
top 在顶部展示内容面板,直接写在 控件属性上 Boolean -- --
slot使用
name 说明
header 头部标题栏,
无 详细内容面板
