带搜索框的

          

        <lr-list
        search
        :items="listData"
        v-on:item="itemClick"
        v-on:item-icon="itemIconClick"></lr-list>
props参数说明
参数 说明 类型 可选值 默认值
items 列表数据 array -- --
search 搜索框是否显示,直接写上就行 boolean -- false
事件说明, 在非作用域插槽 的情况下可用
事件 说明 回调参数
item 当前选中行的左侧文字的事件 返回当前选中行的数据
item-icon 当前选中行的右侧图标的事件 返回当前选中行的数据
slot使用
name 说明
无 直接在组件内部写,用于列表的显示
foot 用于显示底部
header 头部显示
